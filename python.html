我的程式作品：Python

◎ Apache + SQLite 簡易產品管理系統
  使用 mod_python 執行 python，將帳密與資料至於 SQLite 中

◎ AWS CodeCommit 不同帳號間自動轉移

◎ AWS CodeCommit 透過 EC2 打包備份至 S3 Bucket

◎ FTP資料拋轉系統：

   (1) 分析CSV 檔案，逐行取值，傳送至 API URL

   (2) 將回傳結果紀錄於 LOG 檔中，並以先進先出方式，保留一定天數

   (3) 回傳結果與CSV 檔，每日發MAIL 至管理員

◎ 透過AWS CLI 判定每日備份狀況，並搭配 Slack，建立監控結果回報機制

◎ 帳單明細自動化系統

   (1) Parse 金額明細 PDF 檔(pdfminer3k)，並轉換成 CSV 檔

   (2) 依照部門，將總金額與原始 PDF 檔，透過郵件寄出

   (3) 將 CSV 檔內容，透過 xlwings 套件匯入 Excel 中

◎ 每日報表自動化

   (1) 透過 selenium 抓取 AWS Daskboard 圖表

   (2) 透過 poplib 抓取郵件內容，並用 PIL 轉成圖片

   (3) 將抓取的圖片，依照不同的專案，拋送於 Confluence
