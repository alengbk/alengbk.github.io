ElasticSearch 告警

定期掃描 ElasticSearch 特定關鍵字數目，分類統計次數，將異常網址透過可連外主機做 DNS解析與 curl 測試
並將結果已附件方式，傳送至 Telegram 告警
Percona Xtrabackup 自動備份還原

於 MySQL slave 主機上，透過 Xtrabackup 進行每日沒單位的遞增備份，並每小時自動還原至備用主機上
以達到緊急狀態時，可即時查詢或替換用途
SSL 憑證產生器

透過 Certbot 套件，批次將免費憑證( Let's encrypt ) 依需求產出，提供 HTTPS 伺服器使用
DB 狀態分析

搭配 ansible 至各類 DB 執行 cli 指令，並透過回傳值判定服務是否正常
服務運維

搭配 Jenkins 對異常服務，進行服務版本判定與重啟，並建立相關確認流程
搭配 AWX 對服務進行更板作業，並建立相關確認流程
定時壓縮服務 Log 檔，模擬 logrotate 方式進行自動化刪除
使用者容量警告

利用repauota指令，抓出每個使用者目前使用容量，並產生一個文字網頁，供管理者從網頁查詢，
同時判斷當使用者容量快到限制值前的警告比例時，發送告知管理者
Openwebmail分信

在新信夾中，保留程式指定最前面的封數，其餘郵件移至新創建的信夾中
定時大量發送通知信程式

在Openwebmail中修改指定的文字檔，利用別名功能，輸入寄件人位置，等待系統Crontab指定發送時間，透過perl發送信件
POP3+SMTP 攻擊防禦

透過系統Crontab指定程式執行的間隔時間，不透過maillog去分析，而是新建一個指定分鐘內的新Log來分析，
來判別是否為有大量 SASL error訊息，再使用IPFW 封鎖採用動態封鎖IP，將 5組 IP 改以先進先出概念，
儲存於 blocklist檔內，並可於程式內自訂 3組固定封鎖IP
資料一步備份與還原工具

透過預先寫好的腳本，將重要的資料備份於指定的目錄或外接硬碟中
當機器損壞或無法開機時，可藉此還原
Rsync轉移工具

即時建立Rsync 分享傳送設定，讓工程師可以透過簡單的介面，將資料即時傳輸到另一台機器
