系統時間校正

針對重要服務(ex.MySQL)，透過 nc 建立的 API 回傳資訊，輔助抓取 Google 網站時間
發現目的主機時間落差超過 30s 時，立即執行 nptedate，並發送 Telegram 報警通知
Apache + SQLite 簡易產品管理系統

使用 mod_python 執行 python，將帳密與資料至於 SQLite 中，並提供網頁輸入與查詢功能
FTP資料拋轉系統

(1) 分析上傳之 CSV 檔案，逐行取值，傳送至 API URL
(2) 將回傳結果紀錄於 LOG 檔中，並以先進先出方式，保留一定天數
(3) 回傳結果與 CSV 檔，每日發送 MAIL 至管理員
帳單明細自動化系統

(1) Parse 金額明細 PDF 檔(pdfminer3k)，並轉換成 CSV 檔
(2) 依照部門，將總金額與原始 PDF 檔，透過郵件寄出
(3) 將 CSV 檔內容，透過 xlwings 套件匯入 Excel 中

每日報表自動化

(1) 透過 selenium 抓取 AWS Daskboard 圖表
(2) 透過 poplib 抓取郵件內容，並用 PIL 轉成圖片
(3) 將抓取的圖片，依照不同的專案，拋送於 Confluence

網頁自動化

(1) 透過 Flask 網頁，將輸入的更板內容，轉成 CD 程式需要資訊

(2) 將 Prometheus Altermanager 警訊，轉換並發送至 Telegram 告警
